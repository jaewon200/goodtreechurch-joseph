# Design

> 요셉부(유치부) 매주 말씀 아카이브. 무드 앵커: **첨부된 히어로 일러스트** — 봄볕 들판에서 예수님이
> 두 아이의 손을 잡고 작은 교회 앞을 함께 걷는, 픽사풍 소프트 3D 렌더의 따뜻한 그림책 한 장.
> **이 이미지가 사이트의 얼굴이자 톤 앵커다.** 색은 이 그림에서 그대로 뽑고, 말씀을 주인공으로 둔다.
> **기쁨은 그림(마당)으로, 경건함은 정성과 절제(종이)로.**

## Concept — 「봄볕 그림책 (Spring-Light Storybook)」

사이트는 **한 권의 그림책**이다. 작은 교회에서 예수님이 아이들과 손잡고 걷는 이야기 —
그리고 매주 우리는 **새 페이지를 펼쳐** 그 주의 말씀을 만난다. 히어로 이미지는 카메라로
헤집고 다닐 무대가 아니라, **봄볕에 펼쳐진 그림책의 표지**다. 그래서 경험은 집에서 소리 내어
읽어 주는 그림책처럼 흐른다 — 표지에서 열려, 빛이 은은히 숨 쉬고 꽃잎이 흩날리다,
스크롤하면 말씀이 고요히 쉬는 **종이 페이지**로 넘어간다.

**두 개의 면(surface)** 으로 기쁨과 경건을 나눠 담는다 — 이미지 자체의 구조 그대로:

- **마당 면 (Meadow):** 히어로 · 환영 · 부서 소개 · 장식. *기쁨*을 담당한다. 봄볕 색, 흩날리는 꽃잎,
  손글씨 톤, 둥근 형태. 첫 방문자가 "따뜻하고 정성스럽다"를 느끼는 자리.
- **종이 면 (Paper = 성소):** 모든 말씀. *경건과 가독성*을 담당한다. 따뜻한 아이보리 종이,
  넉넉한 세리프 말씀, 짙은 먹, 정적. 장식이 본문을 침범하지 않는다.

**기쁨이 말씀을 감싸고, 말씀은 고요 속에 쉰다.** 이미지가 하는 일과 똑같다. 이 한 문장이 전체
설계 원칙이며, 두 실패 모드(원색 유아틱 / 딱딱한 게시판)를 동시에 피하는 장치다. 화려함은 소프트하게
절제되고(유아틱 회피), 말씀은 손으로 만든 듯 따뜻한 종이 위에 앉는다(게시판 회피).

light 테마. 물리적 장면 한 문장: *"주일 오전, 햇살 좋은 마당에서 부모와 아이가 폰으로 이번 주
말씀을 함께 읽는다."* → 밝고, 따뜻하고, 또렷해야 한다.

## Color

OKLCH. 색은 **히어로 이미지에서 직접 추출**한다. 이전 팔레트는 "어두운 하늘→풀밭 3D 그라데이션 +
골드 태양 + 크림슨 무게"를 전제했지만, 이 이미지는 **크림 지배 · 따뜻한 한낮 · 더스티로즈+겨자골드+봄연두
악센트 · 코랄 포인트 · 따뜻한 우드**다. 가장 큰 변화:

**크림슨은 강등된다.** 이미지엔 크림슨(리추얼/무거운 적색)이 없다. 붉은 것은 하트·손글씨의 작은
**코랄 포인트**뿐. 그래서 크림슨을 시스템 색에서 빼고 — 경건의 "무게"는 **따뜻한 우드/먹**이 잇고,
애정·인터랙션의 주 악센트는 **더스티 로즈**, 햇빛·강조는 **겨자 골드**, 코랄은 *하트에만* 드물게 쓴다.

```css
:root {
  /* ── 종이 면 (말씀이 쉬는 성소) ── */
  --paper:       oklch(0.985 0.010 82);  /* 따뜻한 아이보리 — 페이지·카드 바탕(로브·교회벽 톤) */
  --paper-warm:  oklch(0.966 0.015 80);  /* 살짝 깊은 종이 — 인용 웰, 교대 행 */
  --paper-edge:  oklch(0.930 0.016 82);  /* 실선 경계·구분선 */
  --ink:         oklch(0.260 0.023 60);  /* 말씀 본문 — 따뜻한 near-black, on --paper ≈ 12:1 */
  --ink-soft:    oklch(0.405 0.029 60);  /* 보조 텍스트(주차·날짜) ≈ 6.5:1 */
  --ink-faint:   oklch(0.540 0.029 60);  /* 라벨/메타, 종이 위 ≥16px에서만 ≈ 4.6:1 — 본문 금지 */

  /* ── 마당 면 (기쁨의 세계) ── */
  --sky:         oklch(0.885 0.058 233); /* 소프트 하늘 — 히어로 상단·섹션 필드 */
  --sky-soft:    oklch(0.950 0.028 231); /* 옅은 하늘 — 기본 섹션 배경 워시 */
  --meadow:      oklch(0.785 0.120 143); /* 봄 연두 — 악센트·아카이브 연결선·잎 모티프 */
  --meadow-deep: oklch(0.575 0.112 146); /* 잎·줄기, 초록 텍스트(종이 위) ≥4.5:1 */
  --blossom:     oklch(0.880 0.045 355); /* 더스티로즈 워시 — 부드러운 채움(벚꽃·핀afore 톤) */

  /* ── 따뜻한 앵커 & 악센트 ── */
  --wood:        oklch(0.580 0.065 58);  /* 테라코타/우드 — 팻말·플라크·구조적 따뜻함 */
  --wood-deep:   oklch(0.440 0.060 55);  /* 우드 텍스트/경계, 푸터 바닥(그 위 --paper ≈ 9:1) */
  --rose:        oklch(0.720 0.100 16);  /* 더스티 로즈(연) — 채움·밑줄·하트 하이라이트 */
  --rose-deep:   oklch(0.530 0.130 16);  /* 로즈(진) — 주 인터랙션(링크·CTA·focus·출처) ≥5:1 */
  --gold:        oklch(0.815 0.130 82);  /* 겨자 태양 — 하이라이트·글로·"이번 주" 태그 채움 */
  --gold-deep:   oklch(0.600 0.120 74);  /* 골드 텍스트(종이 위) ≥4.5:1 */
  --coral:       oklch(0.630 0.170 30);  /* 코랄 포인트 — 하트·유일한 CTA에만(아껴서) */
  --coral-ink:   oklch(0.500 0.165 30);  /* 코랄 텍스트가 필요할 때 ≥4.5:1 */

  /* ── 신호 & 재질 ── */
  --halo:        oklch(0.930 0.075 88 / 0.55);          /* 따뜻한 햇무리 radial — 장식 전용 */
  --shadow-warm: 0 6px 24px oklch(0.560 0.075 55 / 0.14); /* 유일한 그림자 토큰(낮고 따뜻) */
  --radius-card: 20px;
}
```

**대비 규칙(하드 제약 — 접근성 척추):**

1. **말씀 본문 및 읽는 길이의 한글:** 항상 `--ink` on `--paper` (≈12:1). 읽어야 하는 텍스트는
   `--ink-soft`보다 옅어질 수 없다. **말씀은 하늘·풀밭·이미지 위에 절대 얹지 않는다 — 오직 종이 카드 위.**
2. **보조 텍스트**(주차·날짜·출처 캡션·부서 본문): `--ink-soft`(≈6.5:1) 이상. 최소 **≥4.5:1** 강제.
3. **`--ink-faint`** 는 본문이 아니다. 종이 위 ≥16px 라벨/장식에만. 말씀·묵상·출처에는 금지.
4. **색 텍스트는 색 필드 위에 두지 않는다.** 로즈/골드/우드/코랄 텍스트는 종이/아이보리 위에서만, 단어를
   담을 땐 `-deep` 변형을 쓴다. 하늘·풀밭 필드 위 텍스트는 종이 칩에 얹거나 `--ink`를 쓴다.
5. **인터랙션 상태:** `--rose` → `--rose-deep`. focus ring = `--rose-deep` 2px + 2px 종이 오프셋(모든 면에서 보임).
6. **골드 주의:** `--gold`는 밝다 — 채움·글로·(짙은 ink 얹은)태그에만. 흰 위 골드 텍스트 금지(`--gold-deep` 사용).
7. **어디에도 풀채도 원색(빨강·노랑·파랑) 없음.** 위 악센트는 설계상 자연 채도 파스텔(로즈/골드 chroma ≤0.13,
   초록 ≤0.115, 작은 코랄 포인트만 0.17까지).

## Typography

**한글 우선. 세 개의 얼굴, 세 개의 역할.** 이미지의 서명은 손으로 그린 "예수님 / 사랑해요" 팻말 —
둥글고 정성스럽다. 이 따뜻함이 타이포에 들어와야 이미지의 혼을 잃지 않는다. 하지만 "전부 둥글게"(유아틱)도,
"전부 명조"(딱딱한 게시판)도 거부한다.

- **디스플레이 / 브랜드 / 섹션 제목 (소리 내어 읽는 목소리):** **둥근 손글씨 계열 한글 디스플레이체.**
  1순위 `강원교육새음체`(무료·둥글·따뜻·큰 사이즈에서 또렷). 셀프호스팅이 번거로우면 `Jua`(Google Fonts)로
  대체. 팻말 목소리 — 요셉부 워드마크, 히어로 카피, 섹션 타이틀에만. **본문/메타/말씀엔 쓰지 않는다.**
  (이상적으로는 "요셉부" 워드마크만 손그림 SVG 락업으로 — 실제 팻말을 오마주.)
- **말씀 본문 (주인공 콘텐츠):** **세리프.** 말씀엔 무게와 정적이 필요하고, 세리프는 경건하며 "구별됨"을
  준다. **`Gowun Batang`(고운바탕 — 부드럽고 둥근 명조, 소프트 렌더 톤과 맞고 어르신 가독성 우수)** 1순위,
  `Nanum Myeongjo` 폴백(폰트 로드 리스크 헤지). **weight 400** — 경건함은 세리프+여백에서 오지, 굵기에서 오지 않는다.
- **UI / 본문 / 메타 / 부서 정보:** **`Pretendard`.** 중립적, 최상의 한글 가독성 — 아이·어르신 접근성을 책임진다.

> 왜 말씀은 둥글게 안 하나? 말씀은 *놀이방에서 구별되어야 하는* 유일한 자리 — 귀엽기보다 거룩해야 한다.
> 둥근 얼굴이 감싸고 환영하면, 명조가 거룩함을 붙든다. 이 의도된 대비가 곧 디자인이다.

```css
--font-display: "강원교육새음체", "Jua", "Pretendard Variable", sans-serif; /* 워드마크·히어로·섹션 제목 전용 */
--font-verse:   "Gowun Batang", "Nanum Myeongjo", serif;                    /* 말씀 본문·큰 인용 */
--font-sans:    "Pretendard Variable", Pretendard, system-ui, -apple-system, sans-serif; /* UI·본문·메타 */
```

**스케일**(모바일 우선, rem @16px, 본문 하한은 의도적으로 높음):

| 역할 | 얼굴 | 크기(모바일 → 데스크톱) | 두께 | 비고 |
|---|---|---|---|---|
| 히어로 워드마크 요셉부 | display | `clamp(2.6rem, 11vw, 4.5rem)` | 700 | text-wrap: balance |
| 히어로 서브라인 | display | `clamp(1.15rem, 4.5vw, 1.6rem)` | 500 | 짧고 따뜻하게 |
| 섹션 타이틀 | display | `clamp(1.6rem, 6vw, 2.4rem)` | 700 | 우드/종이 플라크 위 |
| **말씀 본문** | **Gowun Batang** | **`clamp(1.35rem, 5.2vw, 1.9rem)`** | 400 | **line-height 1.85; 페이지의 시각적 정점** |
| 말씀(이번 주) | Gowun Batang | `clamp(1.6rem, 6vw, 2.15rem)` | 400 | lh 1.8, featured |
| 출처(ref) | Gowun Batang | `1.0rem → 1.05rem` | 600 | `--rose-deep`, 우측 정렬 |
| 묵상(reflection) | Pretendard | `1.125rem → 1.2rem` | 400 | `--ink`, 아이 눈높이 한 줄 |
| 본문 / 부서 정보 | Pretendard | **`1.0625rem(17px) → 1.125rem(18px)`** | 400 | **하한 17px** |
| 주차 / 날짜 메타 | Pretendard | `0.9375rem(15px)` | 500 | `--ink-soft`, tracking +0.02em |
| 내비 / 버튼 | Pretendard | `1.0rem` | 600 | — |

**줄바꿈 & 리듬:** h1–h3·히어로 `text-wrap: balance`; 말씀·문단 `text-wrap: pretty`(한글 orphan 방지).
말씀 측정폭 **한글 ~20–24자/줄**(`max-width: ~24ch`) — 성경은 소리 내어 읽듯 짧고 숨 쉬는 줄을 원한다.
**모든 한글 `word-break: keep-all`**(어절 중간 끊김 금지), 긴 ref/URL만 `overflow-wrap: anywhere`.
디스플레이/제목만 `letter-spacing: -0.01em`, **말씀은 letter-spacing 0**. 문단 간격 넉넉히(0.9em) — 어르신 배려.

## Signature Motifs & 절제 규칙

**일러스트가 시끄러운 만큼 인터페이스는 조용하다.** 모티프는 이미지에서 뽑되 **팔레트 톤의 소프트 SVG**로
그린다 — 포토리얼도, 클립아트도 아니게. **하드 캡: 뷰포트당 장식 모티프 최대 2개, 각 화면 면적 ≤~6%,
모두 `aria-hidden`, 모두 reduced-motion에서 정지.**

| 모티프(이미지) | UI에서 | 위치 / 방식 | 하드 한계 |
|---|---|---|---|
| **손잡기**(인도·소속) | 아이콘 아님 — *서사의 척추* | 레이아웃으로: 아카이브를 따라 내려가는 얇은 `--meadow` **연결선/길**로 표현. 손 그래픽 없음 | 연결선 1개 |
| **하트**(사랑, 교회벽+팻말) | 브랜드 러브마크 & **코랄이 등장하는 유일한 곳** | 교회 로고 마크 안 작은 하트; 묵상 줄 앞 하트 1개 | 화면당 코랄 하트 ≤1, ≤20px |
| **손글씨 나무 팻말** | **제목 플라크 + 로고 크래프트** | 섹션 타이틀이 소프트 우드 플라크(둥근, `--wood` 페인트 테두리, 미세 그레인, 모서리 못 점) 위에 | 플라크는 섹션 타이틀·히어로만, 서브헤더 금지 |
| **벚꽃/데이지 꽃잎** | 앰비언트 패럴럭스 딜라이트 | 히어로·섹션 전환에 아주 느린 소프트 SVG 꽃잎; 데이지는 아카이브 끝 종결 마크 | 화면 꽃잎 ≤5, opacity ≤0.5, 느리게 |
| **나비** | 마이크로 리워드 | "이번 주" 말씀 옆에 로드 시 *한 번* 살포시 앉고(부드럽게) 멈춤. 루프 금지 | 1마리, 비루프 |
| **작은 교회(십자가+하트)** | **브랜드 마크 / 홈** | 단순화한 선+면 채플(크림 벽, 테라코타 지붕, 십자가, 작은 하트) = 로고·푸터·파비콘·"홈" | 하나의 일관된 글리프 |
| **햇무리 / 구름** | 따뜻함 & 은은한 후광 | 히어로와 "이번 주" 카드 뒤에만 소프트 `--gold` radial 글로; 히어로 하늘에 느린 구름 1–2 | 글로는 히어로+이번주만 |

**Anti-slop 가드레일:** 아카이브에 균일 카드 그리드 금지(§Components). 섹션 배경에 일반 linear-gradient 금지 —
소프트 *radial* 따뜻한-빛 워시와 플랫 색 필드만. 그림자 스택 금지, `--shadow-warm` 하나만. 애니메이션 장식은
꽃잎/나비/구름뿐 — 세 번째가 당기면 지운다. 모든 모티프는 팔레트 톤만(순백·순흑·#f00 금지).
**읽는 종이 칼럼은 장식이 없다**(featured 말씀의 닫는 하트 1개만 예외). 이것이 "고급스럽게 귀여움"을 지키고
유아틱으로 미끄러지는 걸 막는다.

## Components

### 말씀 카드 (Verse Card) — 원자 단위, 곧 제품
`--paper`; radius `--radius-card`(20px); `--shadow-warm`; 1px `--paper-edge` 실선; padding `clamp(20px,6vw,36px)`.
**측면 컬러 스트라이프 금지**(게시판 룩). 구조(위→아래):

1. **주차**("2026년 6월 셋째 주") — Pretendard 15px `--ink-soft`, tracking +0.02em, 앞에 우드 톤 점 하나.
2. **말씀** — Gowun Batang, 주인공. `--ink` ≈12:1. 짧은 줄. 카드를 지배한다.
3. **출처**(요한복음 3:16) — Gowun Batang 600, `--rose-deep`, 우측 정렬, 위에 짧은 `--meadow` 룰.
4. **주제** — 작은 둥근 **우드 플라크 태그**(미니 팻말), display 얼굴, 우드 채움 위 `--ink`.
5. **묵상** — `--coral` 하트 글리프 1개 + Pretendard 한 줄, 살짝 `--paper-warm` 웰 안("아이와 함께 나눠요").
- **이번 주 변형(featured):** 크게; 뒤에 `--gold` 후광 글로; "이번 주 말씀" 골드 태그(짙은 ink); 로드 시
  나비가 여기 앉음; 카드를 닫는 중앙 코랄 하트 1개.

### 히어로 (Hero) — 이미지가 주인공
- **풀블리드 일러스트**(§Assets) 상단 ~88svh(svh — 모바일 URL바 안전). `object-fit: cover`,
  초점은 잡은 손/예수님 얼굴(`object-position`으로 세 인물+교회가 프레임에 남게).
- **레이어링(평탄화 금지):** 뒤 `--gold` radial 글로 → 일러스트 → 앞 패럴럭스 꽃잎. **워드마크는 우드 팻말
  플라크**(요셉부 + 서브라인 "예수님과 함께 걷는 우리")로 좌하단 — 이미지 속 실제 팻말 위치를 오마주.
  카피/워드마크는 이미지 픽셀에 기대지 않고 **크림 backdrop-blur 플라크**(`--paper` 0.82 alpha) 위에 얹어
  대비를 보장. `--halo`는 플라크 뒤에, 맨 텍스트 뒤엔 금지.
- **하단 경계 = 소프트 종이 마스크**(찢긴 종이/부드러운 물결, `--paper`) — 일러스트가 첫 종이 페이지로
  *녹아들며* "책이 펼쳐지는" 순간. 딱딱한 크롭 라인 금지.
- 스크롤 큐: 작은 `--rose` 아래 셰브런 + "말씀 보러 가기"(reduced-motion: 정지).
- **히어로에 말씀을 넣지 않는다** — 히어로 = 환영·감정만. 말씀은 아래 자기 섹션에서 조용히 자격을 얻는다.

### 내비 (SiteNav)
미니멀 상단 바, 히어로 위 투명 → 스크롤 시 `--paper`/blur 배경 + `--shadow-warm`.
좌: 채플 로고 마크 + 요셉부(display). 우(원페이지라 햄버거 불필요): "말씀 보기" 단일 텍스트 링크가 아카이브로
부드럽게 스크롤. 높이 56px, safe-area-inset 대응, sticky `z: var(--z-nav)`, focus-visible 링.

### 아카이브 (VerseArchive) — 균일 그리드 아님(의도적 anti-slop)
이미지의 "함께 앞으로 걷기"를 **내려가는 길(path) 타임라인**으로. 얇은 `--meadow` **연결선**이 위→아래로 흐름.
- **이번 주:** 위 featured 말씀 카드, 풀폭, 후광. (아카이브 리스트 안이 아니라 자기 "이번 주 말씀" 섹션에.)
- **지난 말씀:** 길을 따라 스케일이 한 단계씩 내려가는 *컴팩트* 카드(주차 + 말씀 한 줄 미리보기 + 출처).
  탭하면 전체 카드로 펼침(아코디언) 또는 `#YYYY-MM-DD` 앵커로 딥링크. 최신→오래된 순 하향.
- **종결:** 길 끝에 작은 데이지 + "요셉부의 첫 말씀부터 함께해요". 지금 4개라도 비어 보이지 않고 *더 채우자*고 초대.
- 데스크톱(항목 ~6개 초과 시): 길을 중심으로 카드가 좌/우로 *살짝*(≤80px) 어긋나는 징검다리 배치.
  모바일은 항상 단일 칼럼 가독. **`auto-fit` 균일 타일 금지.**

### 푸터 / 부서 정보 (DeptInfo + SiteFooter)
- **부서 정보**: `--sky-soft`(또는 `--meadow`→`--sky-soft`) "봄 마당" 밴드 위, 종이 카드로 콘텐츠를 얹음.
  요셉부 2–3문장 따뜻한 소개(누가·몇 살·언제 모이는지) 존댓말, 채플 마크, 예배 시간·장소는 표가 아닌 소프트 카드.
- **푸터**: **`--wood-deep`** 바닥에 안착 — 긴 밝은 페이지 끝의 유일한 짙고 아늑한 앵커("작은 교회로 돌아오는" 느낌).
  텍스트 `--paper`(≈9:1). 채플 마크 중앙, "좋은나무교회 요셉부"(교회명은 실제로 확인 후), 닫는 한 줄
  "매주 주일, 말씀이 한 편씩 쌓입니다.", 저작권 메타 `--ink-soft`, 작은 하트 1개. 링크 농장 금지.

### 장식 레이어 (DecorLayer)
단일 `<DecorLayer aria-hidden>` 컴포넌트가 꽃잎/구름/나비를 관리 — `prefers-reduced-motion`을 읽고 §Motifs의
카운트 상한을 강제. 그 외 모티프는 토큰 색 인라인 SVG.

## Layout — 섹션 흐름 (모바일 우선, 세로 스크롤)

원페이지. "마당 → 성소 → 마당" 하강:

1. **내비** (sticky, 투명→종이)
2. **히어로** — 풀블리드 이미지 + 팻말 워드마크 + 하단 종이 마스크. *(표지 — 기쁨)*
3. **환영/소개** — `--sky-soft` 필드 위 짧은 따뜻한 문단, 우드 플라크 제목 "요셉부에 오신 걸 환영해요". *(안으로 들어섬)*
4. **이번 주 말씀** — featured 말씀 카드 + 후광 + 나비. 페이지의 경건한 정점. *(성소의 고요)*
5. **지난 말씀** — 내려가는 길 타임라인, 종결 데이지. *(쌓여 가는 기록)*
6. **요셉부 이야기** — 봄 마당 밴드, 소개 + 모임 정보 카드 + 채플 마크. *(다시 소속으로)*
7. **푸터** — 우드-딥 홈 그라운드, 닫는 한 줄.

칼럼: 단일, 읽기 섹션 `max-width: 40rem`(말씀 측정폭 별도), 중앙 정렬, 좌우 여백 `clamp(1.25rem, 5vw, 2rem)`.
세로 리듬: 섹션 padding-block `clamp(3.5rem, 12vw, 6rem)`. Flexbox 1D 기본, (향후) 징검다리 아카이브만 Grid.
**z-스케일:** `decor(0) < field(1) < paper-card(2) < nav(30) < modal(40) < toast(50)`. (캔버스 레이어 없음 — 3D 제거.)

## Motion

**2D 일러스트에 맞춰 재구성**(3D 씬 아님). 은유: *봄볕 속에서 조용히 숨 쉬는 그림책을 한 장씩 넘겨 읽는다* —
스크럽 카메라도, orbit도 없다. GSAP ScrollTrigger는 가볍게, 값싸게 — 모바일 성능과 reduced-motion에 유리.

- **히어로 — 레이어 패럴럭스 + 느린 Ken-Burns:** 일러스트가 아주 느린 Ken-Burns(scale 1.0→1.06, ~18s,
  ease-in-out 왕복). 3개 깊이 레이어(하늘/구름 최저속, 인물 중간, 앞 꽃잎 최고속) 스크롤 패럴럭스, 이동 ≤~40px.
  `--gold` 글로 opacity 은은히 맥동(0.5→0.7, ~6s), 구름 1–2개 아주 느리게 가로 드리프트, 꽃잎 느리게 하강(≤5).
- **스크롤 등장:** 종이 카드 `fade + translateY(16px→0)`, stagger 80ms, `ease-out-expo`(`cubic-bezier(0.16,1,0.3,1)`),
  500–650ms. **바운스·일래스틱 금지.** 히어로의 종이 마스크가 2섹션 진입 시 "펼쳐진다".
- **이번 주 카드:** 최초 진입 시 `--gold` 후광이 한 번 피어나고(700ms) 나비가 한 번 앉아 멈춤 — 유일한 딜라이트 비트.
- **아카이브 길:** `--meadow` 연결선이 스크롤에 따라 그려진다(SVG stroke-dashoffset) — "함께 걷는 길"을 한 번 긋기.
- **마이크로:** 링크/버튼 `--rose-deep` 색 전환 + 1px lift(180ms), 내비 배경 크로스페이드 인.

**`prefers-reduced-motion: reduce`(하드 요구):** Ken-Burns·패럴럭스·꽃잎·구름·나비·선 그리기 **전부 없음.**
모든 요소는 최종 상태로 렌더, 허용되는 전환은 카드 등장의 부드러운 **opacity 페이드(≤200ms)** 뿐.
스크롤 연동 transform 0, 루프 모션 0 → 멀미 없음. 구현: (a) globals.css 블랭킷 규칙 유지 + (b) JS 가드
`if (matchMedia('(prefers-reduced-motion: reduce)').matches) return;`로 **ScrollTrigger 타임라인 자체를 생성하지 않음.**
이건 열화 경로가 아니라 *기본 품질* 경로다.

## Hero Image Usage & Assets (3D 없음)

**히어로 이미지 = 책이 펼쳐지는 표지, 풀블리드.**
- 상단 ~88svh 풀블리드, `object-fit: cover`, 초점 잡은 손/얼굴. 프레임/컷아웃 금지 — 카드에 넣으면 썸네일처럼
  보이고, 오려내면 교회+팻말+봄 배경(브랜드를 carry)을 잃는다. **통째로** 쓴다.
- **배송:** `next/image` `priority`(LCP 요소) + `fill` + `sizes="100vw"`, 또는 `<picture>`로 **AVIF+WebP**
  반응형 `srcset`(768/1200/1600/2000w). **blur-up placeholder**(작은 base64, `--sky-soft` 도미넌트) — 첫 페인트가
  흰색이 아닌 따뜻함으로. `<link rel="preload" as="image">`. **모바일 세로 크롭**(세 인물+교회에 더 타이트) 별도 제공 —
  폰에서 얼굴이 커야 하고 감정 페이로드는 얼굴+잡은 손이다. **목표: 모바일 폭에서 히어로 소스 ≤~250–400KB.**
- **대비 안전:** 워드마크/서브라인은 크림 backdrop-blur 플라크 위 → 이미지 픽셀과 무관하게 대비 확보.
- **alt:** "예수님이 두 어린이와 손을 잡고 작은 교회 앞을 함께 걷는, 따뜻한 봄날의 그림." 장식 레이어 `aria-hidden`.
- **파일:** 히어로를 `public/images/hero.*`에 둔다(`public/images/` 신규 생성).

**3D는 사용하지 않는다.** `public/models/children.glb`(약 502k tris / 19.6MB)는 로드하지 않는다.
이미지가 동일한 감정(예수님+아이들·따뜻함·함께 걷기)을 1/50 무게로, 가독성·성능·reduced-motion 모두 우수하게
전달한다. 두 개의 "히어로" 순간은 그림책의 단일 목소리를 쪼갤 뿐이다.
- **정리(cleanup):** `children.glb`는 삭제 권장(원하면 보관). `package.json`의 `three` · `@react-three/fiber` ·
  `@react-three/drei`는 미사용 → 제거 가능. **`gsap`은 유지**(2D 패럴럭스/스크롤에 사용).

## Voice & Copy

따뜻하고 낮은 목소리. 아이도 알아듣되 가볍지 않게. 말씀이 주인공 — 카피는 말씀을 안내할 뿐 앞서지 않는다.
느낌표 남발·광고체 금지. 존댓말, 짧고 다정하게.
**한 줄 아트디렉션:** *일러스트는 사랑받는 책의 밝고 햇살 가득한 표지가 되게 하라 — 그 표지가 펼쳐지는 모든
페이지는 조용하고 따뜻하고 경건하게 만들어, 부모와 아이가 말씀을 함께 읽는 동안 품에 안긴 듯 느끼게.*

## Accessibility & Inclusion

- 모바일 우선. 터치 타깃 ≥48px, 세로 스크롤 흐름 자연스럽게, safe-area-inset 대응.
- 본문 대비 규칙(§Color) 준수 — 말씀 ≈12:1, 보조 ≥4.5:1. 밝은 배경 위 흐린 회색 금지.
- 다양한 연령(아이·부모·어르신)이 함께 보므로 본문 하한 17px, 한글 가독성 좋은 서체, `keep-all` 줄바꿈.
- 스크롤/패럴럭스 모션에 `prefers-reduced-motion` 페이드-온리 경로 필수(§Motion) — 멀미 방지.
- 저사양 모바일: 히어로 이미지 최적화(AVIF/WebP, 반응형, blur-up, ≤~400KB), LCP 관리. (무거운 3D 없음.)

- **새 서명 시스템: 우드 팻말 플라크**(제목/워드마크) + **채플(십자가+하트) 브랜드 마크** — 이미지에서 직행, "정성" 담당.
- **우드-딥 홈 푸터** 신설(아늑한 짙은 앵커) — 일반 밝은 부서 블록 대체.
- **컨셉 명문화: 「봄볕 그림책」** — 마당(기쁨)이 종이(성소)를 감싸고, 텍스트는 이미지/하늘/풀밭 위에 절대 안 얹음.

**유지(여전히 좋음):** light 테마; 말씀=주인공, 조용한 따뜻한 종이 위; 본문 고대비, 색 필드 위 텍스트 금지;
